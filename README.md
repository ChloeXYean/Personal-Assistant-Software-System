# 🧑‍💻 Personal Assistant App (Python, Tkinter)

This project is a **Personal Assistant software prototype** developed using **Python** and **Tkinter**.  
It was created as part of a group assignment at TARUMT. The application provides three main features to help users organize their daily life.

---

## ✨ Features
- ✅ **To-Do List Manager** (by Jia Min)  
  - Create, organize, and check off tasks  
  - Prioritize tasks with deadlines  

- 📅 **Basic Calendar App** (by Xiao Yean)  
  - Add, view, and manage events  
  - Simple monthly and daily scheduling  

- 💰 **Expense Tracker** (by Jhie Ying)  
  - Log daily expenses and categorize them  
  - View basic summaries of spending  

---

## 🛠 Tech Stack
- **Language:** Python 3  
- **GUI Framework:** Tkinter
